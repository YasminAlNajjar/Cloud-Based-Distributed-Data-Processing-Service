# Cloud-Based Distributed Machine Learning Analytics using Apache Spark

This project implements a cloud-based distributed data processing service using Apache Spark and PySpark on Databricks. The system allows users to upload datasets and perform distributed machine learning analytics and descriptive statistics.

## Technologies Used
- Apache Spark
- PySpark
- Databricks
- Python

## Features
- Upload datasets to cloud storage
- Perform descriptive statistics
- Run multiple machine learning jobs using Spark MLlib
- Simulate cluster scalability and analyze performance

## Dataset
The project uses datasets from the UCI Machine Learning Repository.

## How to Run
- Upload the dataset to Databricks Unity Catalog Volume
- Run the provided notebooks
